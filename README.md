# Streako - Study Planner & Productivity Tracker

**“Build your streak. Beat your distractions.”**

Streako is a premium productivity tool designed specifically for students to encourage consistency, focus, and daily progress.

## Tech Stack

- **Frontend**: React.js, Tailwind CSS (v4), Framer Motion, Recharts, Lucide React
- **Backend**: Node.js, Express.js
- **Database**: MongoDB (Mongoose)
- **Auth**: Google OAuth 2.0 & JWT

## Quick Start

1. **Setup Env**: Configure `server/.env` with your credentials (see [SETUP_GUIDE.md](file:///C:/Users/Jayant/.gemini/antigravity/brain/8e523ac0-9b08-4971-aa3d-eb220d002dec/SETUP_GUIDE.md)).
2. **Install**: 
   ```bash
   npm run install-all
   ```
3. **Run**:
   ```bash
   npm run dev
   ```

## Features

- **Auth**: Secure login with Google or Email.
- **Dashboard**: Visual overview of your progress and tasks.
- **Tasks**: Full control over your study goals with priority levels.
- **Analytics**: Weekly performance charts and streak tracking.
- **UI/UX**: Premium dark theme with glassmorphism and smooth animations.
